Obtaining and Compiling StopWatch

Information on the original StopWatch is available at the World Wide Web page
http://math.nist.gov/StopWatch.

It contains the User's Guide in postscript and html formats, man pages for every StopWatch subroutine, and an overview man page. src contains the source code for the StopWatch module, example programs, and makefiles.
